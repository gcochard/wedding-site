Wedding (or any other event) Website Framework

This project was borne out of frustration with the many free wedding website template systems out there. I wanted something easy to build, fast to iterate, and conducive to complete customization.

This is currently in pre-alpha stage; no security implementations, no session support, and only a few working components. What does work however is fairly polished.
